# Customer Support Chatbot

A simple Customer Support Chatbot built using Python, Flask, HTML, CSS, and JavaScript. The chatbot responds to common customer questions such as greetings, order status, refunds, and payment queries using a FAQ dataset.

## Features

- Simple and user-friendly interface
- Answers frequently asked customer questions
- Reads responses from a CSV file
- Easy to customize
- Built with Flask

## Installation

1. Clone the repository

```
git clone https://github.com/your-username/Chatbot_for_Customer_Support.git
```

2. Open the project folder

```
cd Chatbot_for_Customer_Support
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the application

```
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
